Installation

1
Datei herunterladen
Klicken Sie auf den Download-Button und speichern Sie die Datei KienzleFaktura-Setup-1.0.5.exe.
2
Setup-Datei ausführen
Doppelklick auf die heruntergeladene .exe-Datei. Die Installation läuft automatisch durch — kein weiteres Klicken nötig.
3
App starten — fertig!
KienzleFaktura startet nach der Installation automatisch. Eine Verknüpfung wird auf dem Desktop erstellt. Bereits installierte Versionen werden automatisch ersetzt.
⚠️ Hinweis Windows SmartScreen: Falls Windows SmartScreen beim Ausführen erscheint, klicken Sie auf "Weitere Informationen" und anschließend auf "Trotzdem ausführen". Die Datei ist sicher.
